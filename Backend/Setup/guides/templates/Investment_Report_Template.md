FORMATTING AND STYLE REQUIREMENTS FOR AI AGENTS

- Language: Use UK English spelling exclusively (e.g., capitalise, realise, programme, colour).
- Tone: Maintain a neutral, professional, and analytical tone.
- Data Integrity: If specific data points are unavailable, state "Data Not Disclosed" rather than fabricating figures.
- Conciseness: Avoid redundant phrases; ensure every sentence provides unique value to the investor.
